# Netflix Movies & TV Shows – Exploratory Data Analysis (EDA)

This project explores Netflix movies and TV shows to understand content trends, viewer engagement, and platform-wide insights. Through data cleaning, visualization, and analytical breakdown, the study reveals genre popularity, movie ratings distribution, audience preferences, and yearly content production behavior.

---

## Problem Statement

Netflix offers thousands of titles across various genres and release years. However, identifying the most popular content categories, understanding audience voting patterns, and analyzing production trends are challenging without structured insight.  
This project aims to uncover hidden patterns from Netflix listings to help viewers and creators better understand platform trends.

---

## Objectives

- Analyze genre distribution and identify the most preferred genres
- Examine movie popularity and rating behavior
- Discover movies with highest and lowest popularity
- Evaluate content release trends by year
- Provide actionable insights based on data visualization

---

## Dataset Overview

The dataset includes:

- **Release Date** – Year the movie/show was released
- **Title** – Name of the movie/series
- **Popularity** – Popularity score based on user interactions
- **Vote Count** – Number of audience votes received
- **Vote Average** – Rating score
- **Genre** – Classification of content

A few missing values were found and cleaned during preprocessing.

---

## Workflow

1. Data loading and inspection  
2. Data cleaning:
   - Handling missing data
   - Verifying datatypes and removing unnecessary columns  
3. Exploratory Data Analysis including:
   - Genre frequency analysis
   - Popularity vs genre study
   - Year-wise release trend visualization
4. Insight generation and interpretation  

---

## Key Insights

- **Drama** is the most common genre on Netflix and holds majority audience attention
- **Drama genre movies also receive the highest popularity votes**
- **Spider-Man: No Way Home** is the most popular title in the dataset with Action, Adventure, Sci-Fi genres
- The title with the **lowest popularity** belongs to Music, Drama, History, War, and Sci-Fi genres
- **2020 was the peak year** for content release in this dataset

These insights uncover strong user preference toward drama-based and action-packed titles.
